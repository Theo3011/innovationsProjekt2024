# TutorMatch

## Installation

Følg disse trin for at installere og køre appen lokalt:

1. **Clone repository:**

   ```bash
   git clone https://github.com/Theo3011/innovationsProjekt2024.git
   ```

   ```
   cd innovationsProjekt2024
   ```

   ```
   cd innovationsProjekt
   ```

   ```
    npm install
   ```

   ```
   npm install -g expo-cli
   ```

   ```
   npx expo start
   ```
